# Delivery Spot Map (PWA)

配達員向けに、待機場所・裏道・危険ポイントなどを
地図上にメモとして蓄積できるPWAアプリです。

## Features
- Map-based spot management (Leaflet)
- Add spots by long-press or current location
- Search / filter / sort by distance
- Offline-first (IndexedDB)
- PWA installable

## Tech Stack
- Next.js
- TypeScript / React
- Leaflet
- IndexedDB
- PWA (Service Worker)
